# HomicidiosINEGI
En este repositorio se analizan los microdatos de los registros de mortalidad del INEGI (Estadísticas de Defunciones Registradas) 2022-24


hom.ipynb archivo de limpieza (.dbf a .csv) 

hom22.csv base en formato .csv

hom23.csv base en formato .csv

hom24.csv base en formato .csv

analisis.rmd archivo de análisis y elaboración de visualizaciones

HomicidiosINEGI.pdf visualizaciones y conclusiones sobre los hallazgos
